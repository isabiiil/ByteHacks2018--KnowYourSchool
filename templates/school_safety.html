2010 - 2016 School Safety Report

Summary

This dataset contains information about all criminal incidents in New York City public schools collected by the NYPD. The incidents have been organized by building and school rather than just by school, since a building may contain multiple schools and a school may be in multiple buildings.

Columns

Column Name Definition
School Year Lists the school year in which the data was collected. The most recent year in this dataset is 2015-16.
Building Code A unique code for each building that contains at least one school.
Location Name The name of the organization. Usually, it is the official name of the school.
Location Code A unique code for each school. For schools, it is a combination of the borough code and the school number.
Address The location of the school.
Borough The borough of the location. K stands for Brooklyn, M stands for Manhattan, Q stands for Queens, X stands for Bronx, and R stands for Staten Island.
Geographical District Code The number of the school's geographical district.
Register Number of students on register, or enrolled.
Building Name The official name of the building.
# Schools The number of schools in the building.
Schools in Building Names of the schools in the building. Each school gets its own row.
Major N Number of major crimes in the year.
Oth N Number of other crimes in the year.
NoCrim N Number of non-criminal crimes.
Prop N Number of property crimes.
Vio N Number of violent crimes.
RangeA The building population, given as 1-250, 251-500, 501-750, etc.
NTA The neighborhood of the building.
